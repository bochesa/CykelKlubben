# CykelKlubben
Practice project for Local developer house

This project is meant as a practice project for MVC Asp.Net. The project is a club site for a bicycle club in danish, where the members can see each others equipments and Bicycle adventures.

The project features:
  #Add user
  #User site with user 
  #Add username, telephone number and first- lastname
  #Add bicycle with picture, Model name and gear.
  #Add experience, and add a list of pictures for each experience.
  #About page
  #Members page with list of members (members only)
  #Contact formular
  #Index site includes a carousel slider with three random pictures from members experiences.
  
 Everything is in Danish (sorry)
